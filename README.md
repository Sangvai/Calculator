#  Calculator Application

A clean, responsive, and fully functional calculator web application built with **HTML**, **CSS**, and **JavaScript**. This project demonstrates modern front-end development practices with a sleek dark theme and intuitive user interface.

##  Features

- **Basic Arithmetic Operations** – Addition (+), Subtraction (-), Multiplication (×), Division (÷)
- **Decimal Support** – Handle floating-point calculations accurately
- **Delete Function** – Remove the last entered digit (⌫)
- **Clear Function** – Reset the entire calculation (C)
- **Error Handling** – Prevents division by zero with a user-friendly alert
- **Responsive Design** – Fixed layout that works seamlessly on different screen sizes
- **Visual Feedback** – Button press animations (scale effect)
- **Dark Theme** – Modern, eye-friendly color scheme

##  Live Demo

Open the `index.html` file in any modern web browser to start using the calculator.

##  Technologies Used

- **HTML5** – Semantic markup for the calculator interface
- **CSS3** – Grid layout, custom styling, and animations
- **JavaScript (ES6)** – Core logic, event handling, and DOM manipulation

##  Known Limitations
- Maximum precision is limited to 7 decimal places (rounded)
- Does not support keyboard input (only mouse/touch clicks)
- No support for parentheses or advanced mathematical functions
